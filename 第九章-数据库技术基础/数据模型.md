### 基本概念
  + 概念数据模型
    > 也称信息模型：E-R模型（实体-联系模型）<br>
      用于第一层抽象现实-信息世界，用于数据库设计
  + 基本数据模型
### 数据模型的三要素
  + 数据结构
  + 数据操作
  + 数据约束条件
---
### E-R模型
  + **实体**
    > 用矩形表示<br>
      矩形内写实体名<br>
      区别于其他对象的事件或物体<br>
  + **联系**
    > 用菱形表示<br>
      菱形框中写联系名<br>
      用无向边分别连接有关实体<br>
      用(1:1,1:n,m:n)在无向边上标注联系的类型<br>
    >> 联系的类型：
        一对一：（1:1）
        一对多：（1:n）
        多对多：（m:n）
  + **属性**
    > 用椭圆表示<br>
      实体某方面的特性<br>
    + 属性的分类
      + 简单属性
        > 原子、不可再分<br>
          不特别说明的话，默认为简单属性
      + 复合属性
        > 可以再划分为别的属性
      + 单值属性
      + 多值属性
      + NULL属性
      + 派生属性
