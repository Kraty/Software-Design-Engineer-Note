## UML中的事物和关系
### 事物
  + 结构事物
  + 行为事物
  + 分组事物
  + 注释事物
### 关系
  > 以下4种关系为UML重的**基本关系**，并不是所有关系，还会有很多变体
  + 依赖
  + 关联
  + 泛化
  + 实现
## UML中的图
  > 以下类型的**图**都由自己相关的**事物**和**关系**组成<br>
    是一个由顶点(**事物**)和弧线(**关系**)
### 结构图
  + 类图
  + 对象图
  + 包图
  + 组合结构图
  + 构件图
  + 部署图
  + 制品图
### 行为图
  + 用例图
  + 活动图
  + **交互图**
    + 顺序图(序列图)
    + 通讯图(通信图)
    + 定时图(计时图)
    + 状态图
    + 交互概览图
